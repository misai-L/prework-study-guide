HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown in the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicats how much space we want around the content inside an element.
  • A special wild card symbol, or *, is used to apply a rule to all the elements visible on a page
  • To combine two selectors in to a single CSS rule, use a comma, it allows the declaration to affect both selectors

Git

  • The user should always create a seperate branch feature to work on when they want to make revisions and/or work on bug fixes
  • 'gitrevert' commands are useful for feature corrections.
  • 'gitstatus' helps the user identify what branch they are currently working on.
  • 'git checkout -b feature/add- "add name here"' makes a branch feature to make changes to be later pulled in the main branch.
